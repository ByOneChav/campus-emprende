# Caminos Alternativos

## Registro
- Correo no institucional.
- Correo ya registrado.
- Password invalida.
- Token de verificacion vencido.
- Usuario no verifica cuenta.

## Login
- Credenciales incorrectas.
- Cuenta no verificada.
- Token expirado.
- Backend no disponible.

## Verificacion correo
- Token valido.
- Token vencido.
- Token inexistente.
- Reenvio requerido.

## Servicios
- Datos incompletos.
- Categoria invalida.
- Servicio rechazado por admin.
- Servicio requiere correccion.
- Estudiante elimina servicio.
- Servicio no disponible.

## Solicitudes
- Proveedor rechaza solicitud.
- Cliente cancela solicitud.
- Solicitud duplicada.
- Solicitud expira o queda sin respuesta.
- Servicio ya no disponible.

## Reviews
- Rating invalido.
- Usuario intenta resenar sin solicitud completada.
- Comentario reportado.

## Reportes y administracion
- Reporte sin fundamento.
- Reporte valido.
- Servicio ocultado.
- Usuario reincidente.
- Admin solicita correccion.
- Acceso admin denegado por rol.

## Seguridad
- Usuario sin token.
- Token expirado.
- Usuario con rol incorrecto.
- Endpoint protegido sin autorizacion.
